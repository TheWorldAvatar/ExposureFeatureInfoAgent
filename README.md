# Exposure feature info agent

The purpose of this agent is to display results generated by the [exposure calculation agent](https://github.com/TheWorldAvatar/exposure-calculation-agent). The advantage of this over the standard [Feature-Info-Agent](https://github.com/TheWorldAvatar/Feature-Info-Agent) is that this agent groups the exposure results in a format that is more suited for the calculation results, e.g.

```json
{
    "Count": {"400 m": 10, "800 m": 50},
    "Area": {"400 m": 100, "800 m": 5000}
}
```

which is not possible with the standard feature info agent.

## Routes

1) /feature-info-agent/get

    Mandatory parameter: iri

    Example request (handled by visualisation framework):

    ```bash
    curl http://localhost:3838/exposure-feature-info-agent/feature-info-agent/get?iri=http://subject
    ```

    This is for non-trajectory calculations where results are instantiated as triples.

2) /trajectory/feature-info-agent/get

    Mandatory parameter: iri

    Example request (handled by visualisation framework):

    ```bash
    curl http://localhost:3838/exposure-feature-info-agent/trajectory/feature-info-agent/get?iri=http://subject
    ```

    This is for trajectory based calculations where results are instantiated as time series.

## Build

Populate [docker\credentials](docker\credentials) with `repo_password.txt` and `repo_username.txt`.

Then run the following to build the Docker image:

```bash
docker compose build
```

To push the image to the repository, run the following

```bash
docker compose push
```
